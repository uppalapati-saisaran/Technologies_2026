Lesson 1 : What is Generative AI?
- Generative AI is a type of artifical intelligence that can create new content such as:
  text, images, code, audio and video.
- Instead of only analzing data, it generates new data.
- Example:
  Input - Write a shore story about a robot.
  output - A robot woke up in a slient laboratory...
- The AI genreated new text.

Difference between AI, Machine Learning, and Generative AI

Artifical Intelligence (AI) : AI means machines performing tasks that require human intelligence.
Examples: speech recogination, recommendations systems, self-driving cars.

Machine Learning (ML) : Machine learning is a subset of AI where systems learn patterns from data.
Example : email spam detection 
the system learns from: Spam emails, Non-spam email.
Then it predicts whether a new email is spam.

Generative AI : Generative AI is a type of machine learning that creates new content.
Examples:  chatbots, AI art, code genertion, document summarization 

Real Generative AI Systems:
- Some well-known systems include: ChatGPT, Gemini, Claude 
- This systems are powered by Large Lanaguage Models (LLMs).

what is a Large Language Model (LLM)?
- A Large Language Model is a neutral network tranied on huge amounts of text to understand and generate lanuage.
- Example training data: Books, Websites, Articles, Documentation, Code.
- After training the model can:
  - answer questions, summarize text, translate languages, generate code 

- How LLMs work (Simplied) 
	When you type a prompt: Explain Docker 
	The model predicts the next word repeatedly.
	Example: Docker -> is -> a -> container -> platform -> used -> to -> ... 
	So the model works like a very advanced autocomplete system.

- Important Idea: Tokens
  - LLMs do not read full sentences.
  - They read tokens.
    Example sentence: Docker makes deployment easier.
	Tokens might be: Docker | makes | deployment | easier 
	The AI processes text token by token
