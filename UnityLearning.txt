6.	Common pitfalls & quick fixes
•	Namespace mismatch: ensure using and file namespace match the type locations (you saw this already)
•	Serialized field nulls: prefab/scene references missing in Inspector — validate in Awake/Start
•	Index out-of-range in lists: validate list sizes (example in ResetReelLocations)
•	Excessive allocations in Update — avoid allocating in hot paths
